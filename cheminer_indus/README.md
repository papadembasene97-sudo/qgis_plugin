# CheminerIndus Plugin for QGIS

Un plugin QGIS pour :
- cheminer les canalisations (amont / aval / indus)
- gérer les visites et désélections de branche
- afficher les industriels connectés
- ajouter des photos
- générer un rapport PDF

## Installation

1. Copier ce dossier `cheminer_indus/` dans `~/.qgis3/python/plugins/`
2. Démarrer QGIS et activer le plugin “CheminerIndus”

## Ressources

- `resources_rc.py` est généré depuis `resources.qrc` avec `pyrcc5`
- Les icônes sont référencées `:/icons/...`
